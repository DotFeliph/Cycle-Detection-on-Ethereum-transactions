Detailed Description

Defines the interface for an optimized Wei value parser.

This parser is designed to efficiently convert string representations of numbers (including decimal and scientific notation) into large integer values (mpz_t), which represent the smallest unit of Ether (Wei). It uses a pre-allocated context to improve performance by reusing GMP variables.
